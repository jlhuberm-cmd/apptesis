# 01 · Visión general del proyecto

Plataforma web para evaluar competencias digitales **DigComp 2.2 — Área 4 (Seguridad)**,
competencias 4.1 a 4.4, a partir de CSV exportados de ArcGIS Survey123.

- **Objetivo:** análisis estadístico descriptivo y visualización (radar, barras, distribución).
- **Usuarios:** ADMIN, RESEARCHER, VIEWER.
- **Branding UTPL:** Azul #003B71 · Naranja #F39C12 · Verde #27AE60 · Rojo #E74C3C.
